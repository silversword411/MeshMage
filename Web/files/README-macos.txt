This directory contains the files to join your macOS Intel computer to the
MeshNet.

1. Right-click the nebula file, and select "open" to grant it access to run
   on this machine.
2. Open a terminal and navigate to the directory containing this file.
3. Run sudo ./install.sh

Nebula should now be running on your computer and you should be able to
connect to other machines over the MeshNet.
